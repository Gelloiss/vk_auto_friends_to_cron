Список необходимых настроек для работы скрипта
1) В файле get_token.php на 3 строке вставьте свое id приложения
2) В файле script.php на 2 строке укажите задержку в секундах, в таком порядке (от, до). Например, задрежка 100-200 секунд на это строке должно быть написано: $timing = mt_rand(100,200);
3) В файле db.php на 2 строке укажите данные от базы данных в таком порядке: хост, имя пользователя, пароль пользователя, имя базы данных
4) Установите базу данных (файл db.sql) с помощью импорта\экспорта

5) Рекомендуется установить файл script.php на крон, на каждую минуту
6) Файл send.php на крон устанавливайте по надобности
